# 卡淘猎手 (CardHobbyHelper)

卡淘猎手 Pro 是一款专为卡迷打造的卡淘网页端增强浏览器辅助工具。本插件旨在通过优化前端界面的交互体验，帮助用户更高效地浏览和管理关注的卡片。

---

## 🚀 核心功能介绍

本插件主要提供以下三大前端辅助模块：

### 1. 📡 智能雷达 (Radar)
在商品列表页提供强大的关键词过滤与高亮功能。
* **高级语法支持**：使用空格代表“或”，`+` 代表“必须包含”，`-` 代表“必须排除”。例如 `科比 +特卡 -评级`。
* **快捷关注**：在列表页即可一键将目标卡片加入关注库，支持后台同步与静默刷新。

### 2. ⚡️ 批量执行 (Batch)
针对雷达筛选出的目标卡片，提供批量操作面板。
* **统一操作**：支持对勾选的多件商品设定统一预算，系统将自动依次在本地浏览器中为您打开页面并填入金额。
* **防滥用限制**：为保护账号安全，单次操作最多仅允许勾选 18 件商品。

### 3. 🎯 压哨关注 (Sniper)
在“我的关注”列表页为您提供倒计时辅助功能。
* **战前侦查**：系统会在卡片截标前进行本地核准。若此时【当前价 > 您的目标价】则判定超预算放弃；若【当前价 > 您的目标价的1.5倍】则触发熔断，自动为您取消关注以清理列表。
* **最后时刻**：在拍卖结束前 3-4 秒，辅助在本地填入价格并提交。

## 🛠 安装指南

1. 下载本项目的 `.zip` 压缩包并解压到一个固定文件夹。
2. 打开 Chrome / Edge 浏览器，进入扩展程序管理页面（`chrome://extensions/`）。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择您刚刚解压的文件夹即可。
